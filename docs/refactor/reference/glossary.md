# 📖 Glossary: C2 to SOC Terminology

> Terminology mapping between offensive and defensive domains.

## 🔄 Term Mappings

| C2 Term | SOC Term | Notes |
|---------|----------|-------|
| Agent | EDR Agent | Same technology, defensive purpose |
| Target | Endpoint | Protected asset |
| Implant | Sensor | Monitoring component |
| Beacon | Heartbeat | Status check |
| Task | Scan/Response | Detection or remediation action |
| Payload | Response Action | Defensive action |
| Exfil | Telemetry | Data collection for analysis |
| Persistence | Resilience | Maintain monitoring capability |
| Evasion | Evasion Detection | Detect attacker evasion attempts |
| C2 Channel | Management Channel | SOC communication |
| Domain Fronting | (Deprecated) | Not used in SOC context |
| Operator | SOC Analyst | Security operations personnel |

## 📝 Acronyms

| Acronym | Meaning |
|---------|---------|
| SOC | Security Operations Center |
| EDR | Endpoint Detection and Response |
| SIEM | Security Information and Event Management |
| IOC | Indicator of Compromise |
